расширение для браузеров на движке Chromium которое добавляет темную тему для сайтов МЭШ, делая его менее вырвиглазным

# установка
1. скачайте файлы расширения, или клонируйте репозиторий (для linux или wsl) git clone https://github.com/sTorrentMe/MESH_Dark_theme.git
   
   удобнее всего будет скачать со страницы релтза https://github.com/sTorrentMe/MESH_Dark_theme/releases/tag/1.0

   скачайте архив release-XX.zip (только там все файлы) (XX это версия, может отличатся)
   
   распакуйте архив в любую папку, главное чтобы все файлы расширения были в одной папке
   
3. откройте браузер и в адресной строке введите "chrome://extensions/"

   включите "режим разработчика" (переключатель в правом верхнем углу)
   
   нажмите "загрузить распакованное расширение"
   
   выберите папку куда вы распаковали или клонировали архив
   
   и расширение будет установлено, и соответственно появится на панели расширений
   
   при переходе на сайт МЭШ автоматически применится темная тема
   
   живите, да получайте двойки по геометрии.
